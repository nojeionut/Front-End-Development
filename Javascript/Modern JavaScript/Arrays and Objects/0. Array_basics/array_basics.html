Array Basics

Arrays are a special type of object in JavaScript. Arrays are also what we call a data structure, which is just a way of structuring data.

We create a variable and set it to some brackets with some values inside. The values could be of any type. In this case, we have an array of numbers.

const arr = [374, 755, 398, 118, 289];
console.log(arr);

Array Constructor

You can also create an array using the Array constructor:

const arr2 = new Array(1, 2, 3, 4, 5);

Getting Values by Index

arr[0];
arr[0] + arr[3]; // 492

Different Types of Arrays

const fruit = ['apples', 'oranges', 'pears'];
const mixed = [1, 'string', true, null, undefined, [1, 2, 3]];

Array Length

console.log(arr.length); // 5

Mutating Arrays

Adding a new element to the end:

fruit[fruit.length] = 'Peaches';

Changing an existing value:

fruit[1] = 'mangos';

Shortening the array:

fruit.length = 2;
console.log(fruit); // ['apples', 'mangos']
