Object Nesting, Spreading, Methods, and More

So we saw how to create object literals in JavaScript. I want to go a bit deeper and look at nesting, the spread operator, as well as some static methods on the Object constructor.

Creating Objects

First, I want to show you another way of creating objects that you may run into: by using the Object constructor.

const todo = new Object();
// Same as
const todo = {};

// Adding properties
todo.id = 1;
todo.title = 'Buy milk';
todo.completed = false;

Later on, when we get more into OOP, I'll show you how we can create our own object constructors to create multiple instances of the same object. For now, we'll stick with static object literals.

Nesting Objects

We can put objects inside other objects.

const person = {
  address: {
    coords: {
      lat: 42.3601,
      lng: -71.0589,
    },
  },
};

// Accessing nested objects
person.address.coords.lat;

Spread Operator

The spread operator is used to spread object properties into a new object. Let's look at an example:

const obj1 = { a: 1, b: 2 };
const obj2 = { c: 1, d: 4 };

const obj3 = { ...obj1, ...obj2 }; // { a: 1, b: 2, c: 1, d: 4 }

Object.assign()

We can achieve the same result as the spread operator using Object.assign().

const obj3 = Object.assign({}, obj1, obj2); // { a: 1, b: 2, c: 1, d: 4 }

Arrays of Objects

In many cases, you will deal with arrays of objects.

const people = [
  { id: 1, name: 'John' },
  { id: 2, name: 'Sara' },
  { id: 3, name: 'Mike' },
];

// Accessing a property in an object within an array
people[1].name; // Sara

Object Methods

The Object constructor has some helpful methods we can use.

keys()

Get an array of all the keys in an object.

Object.keys(person); // ['name', 'age', 'address']

Getting the Length of an Object

To get the length of an object, use Object.keys() and then the length property.

Object.keys(person).length; // 3

values()

Get an array of all the values in an object.

Object.values(person); // ['John', 30, { address }]

entries()

Get an array of key/value pairs.

Object.entries(person); // [['name', 'John'], ['age', 30], ['address', { address }]]

hasOwnProperty()

Check if the object has a specific property.

person.hasOwnProperty('name'); // true
