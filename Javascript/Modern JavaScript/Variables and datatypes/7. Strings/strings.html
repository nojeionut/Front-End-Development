JavaScript Type Coercion

Implicit and Explicit Type Coercion

JavaScript automatically converts data types when using operators with different types. This is called Type Coercion.

Examples

5 + '5'; // '55' (string)
5 * '5'; // 25 (number)
5 + null; // 5
5 + undefined; // NaN
NaN == NaN; // false
isNaN('Hello'); // true
Number.isNaN('Hello'); // false

Working with Strings

Concatenation

Using the + operator to combine strings:

'Hello' + 'World'; // 'HelloWorld'
const name = 'John';
const age = 30;
'Hello, my name is ' + name + ' and I am ' + age + ' years old';

Template Literals

Introduced in ES6, template literals simplify string concatenation:

const name = 'John';
const age = 30;
`Hello, my name is ${name} and I am ${age} years old`;

String Properties & Methods

Length

const greeting = 'Hello World';
greeting.length; // 11

Changing Case

greeting.toUpperCase(); // 'HELLO WORLD'
greeting.toLowerCase(); // 'hello world'

Character Access

greeting.charAt(0); // 'H'
greeting[3]; // 'l'

Finding Index

greeting.indexOf('o'); // 4

Extracting Substrings

greeting.substring(0, 4); // 'Hell'
greeting.slice(-11, -7); // 'Hell'

Trimming Whitespace

let x = '   Hello World!   ';
x = x.trim(); // 'Hello World!'

Replacing Content

const url = 'https://traversymedia.com';
url.replace('traversymedia', 'google'); // 'https://google.com'

Checking for Substrings

url.includes('media'); // true

Splitting Strings

greeting.split(' '); // ['Hello', 'World']
