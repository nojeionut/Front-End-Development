Type Coercion

Type Coercion is when data types are converted implicitly by JavaScript. Explicit coercion is when conversion is done manually.

Types of Coercion

  • To String
  • To Number
  • To Boolean

Examples of Type Coercion

String Concatenation

5 + '5'; // '55' (string)

The number 5 is coerced into a string due to the + operator being used for concatenation.

Explicit Conversion

5 + Number('5'); // 10

Multiplication with String

5 * '5'; // 25 (number)

The string '5' is coerced to the number 5 because * only works with numbers.

Null Coercion

5 + null; // 5
Number(null); // 0

Boolean Coercion

Number(true); // 1
Number(false); // 0
5 + true; // 6
5 + false; // 5

Undefined Coercion

5 + undefined; // NaN
Number(undefined); // NaN

NaN Behavior

NaN == NaN; // false

NaN is not equal to itself due to how JavaScript handles NaN.

Checking for NaN

isNaN(NaN); // true
isNaN('Hello'); // true

Number.isNaN(NaN); // true
Number.isNaN('Hello'); // false
