Variables

Variables are containers for pieces of data. That data can be one of many different data types. We'll explore data types later, but for now, let's focus on creating and re-assigning variables.

Declaring a Variable

In JavaScript, variables are declared using one of three keywords:

  • var
  • let
  • const
let firstName = 'John';
let lastName = 'Doe';
console.log(firstName, lastName);
let age = 30;
console.log(age);

Variable Naming Conventions

  • Only letters, numbers, underscores, and dollar signs are allowed.
  • Cannot start with a number.

Multi-Word Variables

let firstName = 'John'; // camelCase
let first_name = 'Sara'; // snake_case
let FirstName = 'Tom'; // PascalCase

Reassigning Values

Variables declared with let can be reassigned:

let x = 100;
x = 200;
let score;
score = 1;

const cannot be reassigned or declared without a value:

const x = 100;
x = 200; // Error
const score1; // Error

Constants with Arrays and Objects

While const prevents reassignment, array and object contents can be modified:

const arr = [1, 2, 3, 4];
arr.push(5); // [1, 2, 3, 4, 5]
arr[0] = 200; // [200, 2, 3, 4, 5]

const person = { name: 'Brad' };
person.name = 'John';
person.email = 'john@gmail.com';
console.log(person);
/* {
  name: 'John',
  email: 'john@gmail.com'
} */

Declaring Multiple Values

let a, b, c;
const d = 10, e = 20, f = 30;

Let or Const - Which to Use?

Use const by default for values that won't change. Use let when reassignment is needed.
