Math Object

JavaScript has a built-in object called Math. This object has various methods for mathematical operations, like finding the square root, absolute value, rounding numbers, and generating random numbers.

Exploring the Math Object

console.log(Math);

Common Math Methods

Math.abs()

Returns the absolute value of a number:

Math.abs(-5); // 5

Math.round()

Rounds a number to the nearest integer:

Math.round(5.5); // 6

Math.ceil()

Rounds a number up to the next largest integer:

Math.ceil(5.5); // 6

Math.floor()

Rounds a number down to the nearest integer:

Math.floor(5.5); // 5

Math.sqrt()

Returns the square root of a number:

Math.sqrt(25); // 5

Math.pow()

Raises a number to a specific power:

Math.pow(5, 2); // 25

Math.min()

Finds the smallest of given numbers:

Math.min(5, 10); // 5

Math.max()

Finds the largest of given numbers:

Math.max(5, 10); // 10

Math.random()

Generates a random number between 0 and 1:

Math.random(); // e.g., 0.83929

Generating a random integer between 1 and 10:

Math.floor(Math.random() * 10) + 1; // e.g., 5

Generating a random integer between two numbers (min and max):

const min = 10;
const max = 20;
const x = Math.floor(Math.random() * (max - min + 1)) + min; // e.g., 17
