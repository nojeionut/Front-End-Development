Working with Numbers in JavaScript

Now let's look at numbers and the Math object in JavaScript. Like with strings, when we add a method to a number, a new number object is created, allowing us to call methods on it.

Creating a Number Object

const num = new Number(5);
console.log(num);

Number Methods

toString()

Returns a string representation of the number.

num.toString(); // "5"

toFixed()

Returns a string with a specified number of decimals.

num.toFixed(2); // "5.00"
num.toFixed(1); // "5.0"

toPrecision()

Returns a number with the specified precision.

const num2 = 94.4058;
num2.toPrecision(3); // "94.4"

toExponential()

Returns a string representation in exponential notation.

num.toExponential(2); // "5.00e+0"

toLocaleString()

Returns a string representation of the number in the current locale.

let num = 5000000;
num.toLocaleString(); // "5,000,000"
num.toLocaleString('en-IN'); // "50,00,000"

Number Object Properties & Values

MAX_VALUE

Largest possible value of a number.

Number.MAX_VALUE; // 1.7976931348623157e+308

MIN_VALUE

Smallest possible value of a number.

Number.MIN_VALUE; // 5e-324

isNaN()

Checks if the value is NaN.

Number.isNaN(NaN); // true
